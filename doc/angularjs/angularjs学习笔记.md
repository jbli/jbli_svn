Angularjs 学习笔记

# 安装学习环境
[学习教程](http://docs.angularjs.org/tutorial)

##搭建学习环境
1. 下载angular-phonecat
```
git clone https://github.com/angular/angular-phonecat.git
```
2. 安装node，下载node*.pkg，双击安装
3. npm install karma 
该步安装报错

4 启动
```
cd angular-phonecat
node scripts/web-server.js
```
